# Self JP App

A desktop app for daily Japanese study, built with [Tauri v2](https://tauri.app) + React + TypeScript. Single window with a mode switcher and a compact "bubble" window mode for studying alongside other work.

The UI and all learning content (kanji meanings, vocabulary, grammar explanations) are in Vietnamese, aimed at Vietnamese speakers learning Japanese.

## Modes

- **Kana** — random Hiragana/Katakana flashcards with configurable session length and speed, TTS audio.
- **Kanji (N5–N1)** — daily kanji list sized by a configurable words-per-day setting, per-level browsing, Hán-Việt reading + Vietnamese meanings, example words (from the MNN vocab list, with Vietnamese meanings), daily progress tracking, TTS audio.
- **Từ vựng (Vocab)** — Minna no Nihongo Shokyū I & II vocabulary, browsable lesson by lesson (1–50), Vietnamese meanings, TTS audio.
- **Ngữ pháp (Grammar)** — MNN grammar points by lesson (1–50), each with structure, Vietnamese explanation, and an example sentence with audio.

## Data sources

Generated by [scripts/build-data.mjs](scripts/build-data.mjs) into `src/data/*.json`, committed to the repo (the shipped app never fetches data at runtime). Re-run `node scripts/build-data.mjs` to refresh from upstream (requires `curl`/`unzip` on PATH).

- Kanji + JLPT level: [davidluzgouveia/kanji-data](https://github.com/davidluzgouveia/kanji-data)
- Kanji Hán-Việt readings/meanings: [trungnt2910/KanjiDictVN](https://github.com/trungnt2910/KanjiDictVN) — generator code is MIT-licensed; the Hán-Việt readings and meanings themselves are attributed to hvdic.thivien.net (Từ điển Hán Nôm). Used here for personal, non-commercial study.
- Minna no Nihongo vocabulary structure: [vitto4/MinnaNoDS](https://github.com/vitto4/MinnaNoDS) — **non-commercial use only**; the word selection is 3A Corporation's IP and this dataset is meant to be used *alongside* the official textbooks, not as a substitute. Vietnamese translations of all 2,692 entries are hand-authored in [scripts/mnn-vi.mjs](scripts/mnn-vi.mjs) (a tested automated extraction from a scraped Vietnamese JMdict was too noisy/incomplete for common words, so this was done manually).
- Kanji example words are cross-referenced from the (translated) MNN vocab list rather than a separate untranslated source, so every example word carries a correct Vietnamese meaning for free. Kanji outside MNN's beginner scope (mostly N2/N1) simply have no examples.
- Grammar content ([scripts/mnn-grammar.mjs](scripts/mnn-grammar.mjs)): an original curated summary following the well-known MNN lesson progression — not a transcription of the textbook.
- Kana table (hand-authored, no external source): seion + dakuten + handakuten (142 entries). Combination sounds (yōon) are not yet included.

Audio is OS-native text-to-speech for all modes (`say` on macOS, PowerShell SAPI on Windows) — no bundled audio files ship yet, since no clearly-licensed (CC0/MIT/public-domain) kana audio dataset was found. See `assets/audio/kana/README.md`.

## Development

### macOS

```bash
xcode-select --install
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"

npm install
npm run tauri dev
```

Production build: `npm run tauri build` → `.app` + `.dmg` under `src-tauri/target/release/bundle/`.

### Windows

Prerequisites:
- **Microsoft C++ Build Tools** ("Desktop development with C++" workload)
- **WebView2** runtime (preinstalled on Windows 11; Windows 10 may need the [Evergreen Bootstrapper](https://developer.microsoft.com/microsoft-edge/webview2/))
- **Rust** via [rustup-init.exe](https://rustup.rs) (MSVC toolchain, the default on Windows)

```powershell
npm install
npm run tauri dev
```

Production build: `npm run tauri build` → `.msi` and/or NSIS `.exe` installer under `src-tauri\target\release\bundle\`.

Tauri doesn't cross-compile installers — a Windows build must run on Windows (or CI); this hasn't been verified on Windows yet.

## Known MVP gaps / fast-follows

- Bundled kana audio (currently TTS-only pending a properly licensed dataset).
- Full example sentences (not just isolated example words) for kanji.
- Grammar content covers 1-2 core points per lesson, not exhaustive textbook coverage.
- Yōon (combination sound) kana entries.
- Windows build verification (needs a Windows machine or CI).
