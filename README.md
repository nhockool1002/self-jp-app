# Self JP App

A desktop app for daily Japanese study, built with [Tauri v2](https://tauri.app) + React + TypeScript. Single window with a mode switcher and a compact "bubble" window mode for studying alongside other work.

## Modes

- **Kana** — random Hiragana/Katakana flashcards with configurable session length and speed, TTS audio.
- **Kanji (N5–N1)** — daily kanji list sized by a configurable words-per-day setting, per-level browsing, readings/meanings/example words, daily progress tracking, TTS audio.
- **Vocab** — Minna no Nihongo Shokyū I & II vocabulary, browsable lesson by lesson (1–50), TTS audio.

## Data sources

Generated by [scripts/build-data.mjs](scripts/build-data.mjs) into `src/data/*.json`, committed to the repo (the shipped app never fetches data at runtime). Re-run `node scripts/build-data.mjs` to refresh from upstream.

- Kanji + JLPT level: [davidluzgouveia/kanji-data](https://github.com/davidluzgouveia/kanji-data)
- JLPT vocabulary (used for kanji example words): [Bluskyo/JLPT_Vocabulary](https://github.com/Bluskyo/JLPT_Vocabulary)
- Minna no Nihongo vocabulary: [vitto4/MinnaNoDS](https://github.com/vitto4/MinnaNoDS) — **non-commercial use only**; the word selection and translations are 3A Corporation's IP and this dataset is meant to be used *alongside* the official textbooks, not as a substitute. Keep this app's use non-commercial.
- Kana table (hand-authored, no external source): seion + dakuten + handakuten (142 entries). Combination sounds (yōon) are not yet included.

Audio is OS-native text-to-speech for all three modes (`say` on macOS, PowerShell SAPI on Windows) — no bundled audio files ship yet, since no clearly-licensed (CC0/MIT/public-domain) kana audio dataset was found. See `assets/audio/kana/README.md`.

## Development

### macOS

```bash
xcode-select --install
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"

npm install
npm run tauri dev
```

Production build: `npm run tauri build` → `.app` + `.dmg` under `src-tauri/target/release/bundle/`.

### Windows

Prerequisites:
- **Microsoft C++ Build Tools** ("Desktop development with C++" workload)
- **WebView2** runtime (preinstalled on Windows 11; Windows 10 may need the [Evergreen Bootstrapper](https://developer.microsoft.com/microsoft-edge/webview2/))
- **Rust** via [rustup-init.exe](https://rustup.rs) (MSVC toolchain, the default on Windows)

```powershell
npm install
npm run tauri dev
```

Production build: `npm run tauri build` → `.msi` and/or NSIS `.exe` installer under `src-tauri\target\release\bundle\`.

Tauri doesn't cross-compile installers — a Windows build must run on Windows (or CI); this hasn't been verified on Windows yet.

## Known MVP gaps / fast-follows

- Full example sentences and richer definitions for kanji (currently readings + meanings + matched example words; JMdict/KANJIDIC2 integration is a natural next step).
- Yōon (combination sound) kana entries.
- Windows build verification (needs a Windows machine or CI).
